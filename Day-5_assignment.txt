Goal: Build a React app that fetches a list of users, allows navigation to a detailed profile, and handles side effects correctly.

1. Project Requirements

useEffect Hook: Trigger API calls automatically when components load.
Public API: Use JSONPlaceholder (https://jsonplaceholder.typicode.com/users).
